# 🛰️ STARLINK – Revendeur RDC

A responsive website for selling Starlink internet packages in the Democratic Republic of Congo (DRC), with Airtel Money payment integration.

## 🌐 Live Demo
> https://YOUR_USERNAME.github.io/starlink-rdc

## 📄 Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage — browse all available data plans |
| `payment.html` | Airtel Money payment page with OTP verification |
| `status.html` | Real-time service status indicators |
| `orders.html` | Order history lookup by phone number |

## 💳 Payment Flow

1. Client selects a plan → clicks **Sélectionner**
2. Enters their **Airtel Money phone number** and **PIN**
3. An **OTP verification card** appears with the entered number
4. Client enters the **4-digit OTP code**
5. ✅ Payment confirmed and plan activated
